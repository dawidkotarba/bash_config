# shell_config

Some aliases for zsh shell making my life easier :)

Installation:
1. Clone to home folder
2. Execute setup.sh
3. Ensure the main.sh file is sourced in ~/.zshrc, otherwise add below to ~/.zshrc:
```bash
source ~/shell_config/main.sh
```
